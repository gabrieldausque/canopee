# ConsoleCanopeeLogger class

This is the default logger. Log all message to the console with a color code based on the LogLevel. Configuration will be :

```csharp
{
    ...
   "Canopee": {
   ...
       "Logging": {
               "LoggerType": "Console"
       },
   ...
  }
...
}
```

The LoggerType is Console
